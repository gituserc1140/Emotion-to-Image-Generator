# Emotion to Image Generator

This Streamlit app generates images based on user-inputted emotions using the Replicate API with the SDXL model.

## Setup
1. Install required packages:
   ```bash
   pip install -r requirements.txt
   ```
2. Obtain a Replicate API token from [Replicate](https://replicate.com/).
3. Run the Streamlit app:
   ```bash
   streamlit run app.py
   ```

## Usage
1. Enter your Replicate API token in the sidebar.
2. Input an emotion (e.g., happy, sad, angry).
3. Click "Generate Image" to see the result.
